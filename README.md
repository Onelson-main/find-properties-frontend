This is a simple Next JS app that shows a list of properties near you
## Setting Up

First, install dependencies using:
```bash
npm install
# or
yarn add
```

## Running the App

Start the app using:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
